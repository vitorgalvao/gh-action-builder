# GitHub Action Builder

GitHub Action Builder is a command-line tool to build a sane [GitHub Actions](https://github.com/features/actions) YAML by answering questions.

## Installation

Install with [Homebrew](https://brew.sh):

```shell
brew install vitorgalvao/tiny-scripts/gh-action-builder
```

Alternatively, download the executable at the root of this repository and call it directly.

## Usage

```
Build a sane initial GitHub Actions YAML by answering questions.

Usage:
  gh-action-builder <output-file>

Options:
  -h, --help   Show this help.
```
